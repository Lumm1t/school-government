## Install

`git clone https://github.com/Lumm1t/school-government`

`npm install` or `yarn`

Change environment variables:

1. Rename `.env.example` to `.env`
2. Open `.env` and change `NUXT_ENV_DATOCMS_API_TOKEN` & `GOOGLE_ANALYTICS_ID`.

## Run

### Development

`npm run dev`

### Production

`npm run generate`
`npm run start`
